zkui-generate-all

Purpose

Generates views,composer and a controller for the given domain class

Examples

grails zkui-generate-all Book
grails zkui-generate-all *

Description

Same as Grails's generate-all Command

supports a feature known as scaffolding which involves the generation of a CRUD (Create/Read/Update/Delete) interface for a given domain class.

Usage: 

grails zkui-generate-all [domain class name or wildcard]
