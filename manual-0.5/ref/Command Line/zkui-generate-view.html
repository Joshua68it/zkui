zkui-generate-view

Purpose

Generates views and composer for the given domain class

Examples

grails zkui-generate-view Book
grails zkui-generate-view *

Description

Same as Grails's generate-view Command

supports a feature known as scaffolding which involves the generation of a CRUD (Create/Read/Update/Delete) interface for a given domain class.

Usage: 

grails zkui-generate-view [domain class name or wildcard]
