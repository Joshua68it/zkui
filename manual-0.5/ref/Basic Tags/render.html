render

Purpose

To render a zk component.

Examples

View 

<z:render comp="${window}"/>

Controller 

def index = {
    def window = new Window(title: "My First ZK UI Application", border: "normal")
    window << {
        label(value:"Hello World!")
    }
    [window:window]
}

Description

Attributes

  • comp - zk Component, a instance of org.zkoss.zk.ui.Component
